# Python Practice 🐍

A collection of Python problems and solutions as I learn and practice core Python concepts, logic building, and problem-solving.

## 📚 About

This repo is my personal practice log while learning Python — covering fundamentals, logic-building exercises, and common interview problems.

## 🗂️ Topics Covered

- **Number Problems** — factorial, prime numbers, Armstrong numbers, GCD/LCM
- **String Manipulation** — palindrome check, reverse string, vowel counting, character frequency
- **Data Structures** — working with dictionaries, arrays, NumPy arrays
- **Searching & Logic** — binary search, even/odd checks, age checker
- **Loops & Conditionals** — range-based problems, removing duplicates without built-in functions
- **Series & Sequences** — Fibonacci series

## 📁 Structure

Each file/folder focuses on one problem, with clean, commented code explaining the logic and approach.

## 🎯 Goal

To build strong Python fundamentals for data analysis, automation, and interview prep — one problem at a time.
